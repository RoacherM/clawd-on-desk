Clawd Runtime Flow

Agent events are normalized into a small localhost contract, then resolved by the session state machine and rendered through the active theme.

Pet Shape

  • Theme assets live in theme.json + assets/.
  • SVG states can expose eye-tracking ids.
  • GIF/APNG/WebP/PNG/JPG use the image channel.

Behavior

  • Agent events map to semantic states.
  • State resolver picks the highest-priority live session.
  • Theme tiers choose concrete animation files.

New Agents

  • Implement an adapter that POSTs to /state.
  • Use /permission only if the agent can wait or bridge replies.
  • Add an agent registry entry for settings and labels.
